# Quiz emoções

Quiz de emoções Jogo android. Foi pensado para pais e crianças.

## 📚 Funcionalidades
- Quiz com imagens de emoções
- Várias opções de resposta em cada pergunta
- Ordem aleatória das escolhas
- Pontuação ao longo do jogo
- Ecrã final com mensagem de parabéns

## 🧰 Tecnologias utilizadas
- Java

## ▶️ Como executar
- Android studio.
- Existe release com apk para instalar no telemóvel android.
